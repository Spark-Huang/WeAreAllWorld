## 技术决策

### 数据评级系统
```
📝 普通 (+1) - 日常问候
💬 活跃 (+2) - 日常对话
💜 稀有 (+3) - 情感表达
🧡 珍贵 (+4) - 分享经历
❤️ 典藏 (+5) - 深度思考
💎 绝版 (+6-8) - 特殊回忆
```

### 濒危警告级别
```
🟢 正常 (>30) - 无警告
🟡 注意 (15-30) - 低警告
🟠 警告 (5-15) - 中警告
🔴 危险 (0-5) - 高警告
💀 紧急 (0) - AI即将被回收
```

### 连续登录里程碑
```
🤝 3天 - 忠实伙伴 (+10)
🌟 7天 - 周度伙伴 (+30)
💎 14天 - 长期陪伴 (+50)
🏆 30天 - 月度纪念 (+100)
👑 100天 - 百日知己 (+300)
💫 365天 - 年度灵魂伴侣 (+1000)
```
